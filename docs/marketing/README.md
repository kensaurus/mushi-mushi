# docs/marketing — the Mushi-chan playbook

This folder is the growth kit for Mushi Mushi. It assumes zero budget and one human
(you) with a laptop. The goal is **GitHub stars + npm installs**, the voice is
**Mushi-chan**, and the bar is *nothing in here should sound like it came from a
SaaS deck*.

If any line in this folder could have been written by a generic B2B company, delete it.

## The one-line message (use everywhere, word-for-word)

> **Sentry sees what your code throws. Mushi sees what your users feel.**

Every surface — Show HN title, tweets, LinkedIn, Reddit, dev.to, YouTube description —
opens with some variant of that contrast. Research (AFFiNE, PostHog, Tabularis) is
unanimous: a distinctive, repeatable one-liner compounds. Generic positioning doesn't.

## The one thing *not* to do

Don't chase paid-Cloud conversion yet. The research (AFFiNE, PostHog, Tabularis,
Supabase) is unanimous: **build the trusted devtool brand first, monetize second**.
For the first 90 days, the repo and the SDKs are the product. The hosted tier is
quietly discoverable for those who want it, never pushed.
