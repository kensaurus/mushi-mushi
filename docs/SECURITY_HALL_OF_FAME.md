If you reported a vulnerability and would like to appear here, reply to
the original disclosure thread (or open a follow-up) with:

- Display name (real name, alias, or "anonymous")
- Optional link (your website, GitHub, Twitter / X, Mastodon, BlueSky)
- Confirmation that you're OK with being publicly listed

We will not list anyone without explicit, opt-in consent.

See [`SECURITY.md`](../SECURITY.md) for the full reporting policy and
safe-harbor terms.
