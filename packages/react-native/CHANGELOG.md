# @mushi/react-native

## 0.1.0

### Minor Changes

- Initial public release of the Mushi Mushi SDK platform.
  - `@mushi/core` — universal bug-report engine with structured logging, PII scrubbing, offline queue
  - `@mushi/web` — browser SDK with session replay, console/network/click capture, proactive detection
  - `@mushi/react` — React hooks and provider (`MushiProvider`, `useMushi`, `useMushiReport`)
  - `@mushi/react-native` — React Native SDK with shake-to-report and bottom sheet widget
  - `@mushi/vue` — Vue 3 composables (`useMushi`) and plugin
  - `@mushi/svelte` — Svelte stores and context integration
  - `@mushi/angular` — Angular service and module
  - `@mushi/mcp` — Model Context Protocol server for AI-assisted triage
  - `@mushi/cli` — CLI for project setup and report management
