# Understand your codebase (Codebase Atlas)

Ask natural-language questions about your indexed repository with file citations.

## Steps

1. Ensure GitHub is connected and code is indexed — **Explore** (`/explore`) → **Index** tab.
2. Open the **Ask** tab and type a question like "How does authentication work?"
3. Click citation chips to jump to files in the graph or symbol panel.
4. Use **Tour** for a guided walkthrough of key modules.
5. Use **Search** tab for semantic or symbol-name lookup without chat.

## Tips

- Scope indexing to subdirectories in Index → scope paths.
- Add wiki/knowledge sources in the Knowledge panel for doc merge (when indexed).
