# Set up a project

Get from zero to capturing your first bug report.

## Steps

1. Open **Get started** (`/onboarding`) for the guided three-step wizard.
2. Create a **project** on `/projects` if you skipped onboarding — name it after your app.
3. Go to **Connect & Update** (`/connect`) to connect GitHub, install the SDK snippet, and add MCP to Cursor.
4. Copy the SDK install command and paste it into your app's entry point.
5. Send a **test report** from the Connect page or your running app.
6. Confirm the report appears on **Dashboard** (`/dashboard`) and **Reports** (`/reports`).

## Tips

- API keys and BYOK LLM keys live under **Settings** (`/settings`).
- Switch projects with the project switcher in the top bar — Ask Mushi scopes answers to the active project.
