# Linear plugin

Files a Linear issue every time the classifier categorises a report.

## Setup

1. Create a Linear API key ‚Üí **Settings ‚Üí API ‚Üí Personal API keys**.
2. In Mushi: **Marketplace ‚Üí Linear ‚Üí Install**.
3. Provide:
   - `apiKey` ‚Äî pasted into Vault, never echoed back
   - `teamId` ‚Äî the Linear team to file under
   - `labelMap` *(optional)* ‚Äî JSON object mapping
     `taxonomy_path ‚Üí label_id[]`

## Behaviour

- **`report.classified`** ‚Üí creates a Linear issue with:
  - title = the report title
  - description = Markdown summary + report URL
  - priority = `p0‚Üí1, p1‚Üí2, p2‚Üí3, p3‚Üí4`
  - labels = `labelMap[taxonomy_path] ?? []`
- **`report.status_changed ‚Üí resolved`** ‚Üí moves the linked Linear
  issue to *Done*, leaving a comment with the merged PR URL.
