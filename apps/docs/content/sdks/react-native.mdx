# `@mushi-mushi/react-native`

React Native SDK with shake-to-report (via `react-native-shake`) and a
bottom-sheet capture UI. Reports queue to AsyncStorage when offline.

```tsx
import { MushiProvider, useMushi } from '@mushi-mushi/react-native'
```

See [Quickstart → React Native](/quickstart/react-native).
