# `@mushi-mushi/cli`

Project setup, schema sync, RAG indexing, and report management from the
terminal.

```bash
pnpm add -g @mushi-mushi/cli

mushi login
mushi project create
mushi index ./src                 # build the codebase RAG index
mushi reports list --status new
mushi reports show rep_01HXY…
```
