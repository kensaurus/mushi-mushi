# Quickstart

**Start here if you vibe-code:** the [incident loop](/quickstart/incident-loop) gets you from broken prod to a paste-ready fix prompt in Cursor — usually under a few minutes.

Or run the wizard: `npx mushi-mushi` (detects your framework, writes env vars, optional test report).

## Need a project ID?

Sign up at [app.mushimushi.dev](https://app.mushimushi.dev) (or
[self-host](/self-hosting)) and create a project. Copy env vars from
[`examples/sdk.env.example`](https://github.com/kensaurus/mushi-mushi/blob/master/examples/sdk.env.example).
