## Where do I get my Project ID and API key?

**Option A — zero copy-paste (recommended):**

```bash
npx mushi-mushi login   # or: mushi login
```

The wizard opens your browser, you click **Approve**, then picks or creates a project
and saves the key automatically. Run `mushi init` afterwards to wire the SDK.

**Option B — console:**

Open the [admin console](https://kensaur.us/mushi-mushi/admin) → **Projects** → **New project**.
After creation, the success screen shows both the **Project ID** (UUID) and the **API key**
together with a pre-filled `mushi init` command. The key is shown **once** — copy it now.

**Option C — CI / non-interactive:**

```bash
mushi login --api-key mushi_xxx --project-id 00000000-0000-0000-0000-000000000000
```

The values you need:

| Value | Where to find it |
|---|---|
| `projectId` | UUID on the Projects page — `kensaur.us/mushi-mushi/admin/projects` |
| `apiKey` | Shown once at project creation, or **Projects → Keys → Create key** (scope: `report:write`) |
