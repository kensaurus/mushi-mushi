## 60-second proof

```bash
npx mushi-mushi
```

The wizard auto-detects your framework, installs the right SDK, writes env vars, and prints the snippet. When something breaks: the bug lands in your queue, Mushi produces **the diagnosis** (plain-English root cause + fix), and you pull it into your editor over MCP.

```bash
npx mushi-mushi setup --ide cursor    # then ask Cursor: "what's broken in prod?"
```

**No monitoring stack required.** Self-host in under five minutes, or use the free hosted tier — 50 diagnoses / month, no card.

## What this is (and is not)

- **Built for the solo vibe coder** who ships fast with AI and loses afternoons when prod breaks in code they did not fully write.
- **Editor-native.** The answer comes to Cursor / Claude Code over MCP — not another dashboard to go read.
- **Not an enterprise monitoring stack.** Standalone first; Sentry enrichment is the upgrade reveal, never the front door.
